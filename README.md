1. 파이썬 기본

```python

# 자료형

# 문자열 포맷팅
pi = 3.141592
print(f'파이는 {pi}입니다.')         # 파이는 3.121592입니다.
print(f'파이는 {pi:0.02f}입니다.')   # 파이는 3.14입니다.
print(f'파이는 {pi:10.3f}입니다.')    # 파이는        3.142입니다.
```
